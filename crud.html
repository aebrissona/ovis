¿ Que es el CRUD ?

El concepto CRUD está estrechamente vinculado a la gestión de datos. CRUD hace referencia a un acrónimo en el que se reúnen las primeras letras de las cuatro operaciones fundamentales de aplicaciones persistentes en sistemas de bases de datos:

  • Create (Crear registros)
  • Read o Retrieve (Leer registros)
  • Update (Actualizar registros)
  • Delete 0 Destroy (Borrar registros)

En pocas palabras, CRUD resume las funciones requeridas por un usuario para crear y gestionar datos. Varios procesos de gestión de datos están basados en CRUD, en los que dichas operaciones están específicamente adaptadas a los requisitos del sistema y de usuario, ya sea para la gestión de bases de datos o para el uso de aplicaciones. Para los expertos, las operaciones son las herramientas de acceso típicas e indispensables para comprobar, por ejemplo, los problemas de la base de datos, mientras que para los usuarios, CRUD significa crear una cuenta (créate) y utilizarla (read), actualizarla (update) o borrarla (delete) en cualquier momento (Digital Guide, 2017).

Las instrucciones más habituales son SELECT, INSERT, UPDATE, DELETE. Veamos su sintaxis básica y algunos ejemplos de uso. Para ello utilizaremos una tabla noticias con cinco campos: (González (s.f.)).

Figura 1. Tabla Ejemplo para el manejo del CRUD.

Los campos de la tabla son:

Un identificador único de la noticia, el título de la noticia, el texto de la noticia, la categoría de la noticia y la fecha de publicación de la noticia

Lenguaje SQL

Sentencia SELECT

Sintaxis: SELECT (campo de la tabla) o (*) FROM tabla WHERE (condición);

Ejemplo:

SELECT * from noticias WHERE fecha = NOW;

En este ejemplo estamos pidiendo que nos muestre todas las noticias publicadas el dia de hoy.

Sentencia INSERT

Sintaxis: INSERT [INTO] nombre_tabla [(nombre_columna1),(nombre_columna2)....(nombre_columnaN)] VALUES ((dato de la columna), (dato de la columna)....;

Ejemplo:

INSERT INTO noticias (id, titulo, texto, categoria, fecha) VALUES (37, “Nueva promoción en Nervión”, “145 viviendas de lujo en urbanización ajardinada situadas en un entorno privilegiado”, “promociones”, NOW);

En este ejemplo le estamos diciendo que inserte un registro de acuerdo a los parametros dados.

Sentencia UPDATE

Sintaxis: UPDATE nombre_tabla SET nombre_columna1=(lo que quieras cambiar) [, nombre_columna2=(lo que quieras cambiar) ...] WHERE (condición) ;

Ejemplo:

UPDATE noticias SET categoria = “ofertas” WHERE id=37;

En este ejemplo le estamos diciendo que modifique la cateogira de noticias con el id igual a 37.

Sentencia DELETE

Sintaxis: DELETE (columna) FROM nombre_tabla WHERE (condición);

Ejemplo:

DELETE FROM noticias WHERE fecha = NOW;

En este ejemplo le estamos diciendo que elimine las noticias que hayan sido publicadas hoy.
