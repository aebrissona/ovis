Bienvenidos al OVI de Programación Web

El presente Objeto Virtual de Iformación (OVI), pretende mostrar un contenido explicito y académico sobre los conceptos básicos del curso de programación web 1. El lenguaje PHP como lenguaje principal y la vinculación de un DB a una pagina con PHP.

Objetivos del OVI

  • Mostrar los conceptos basicos del lenguaje PHP
  • Visualizar los pasos necesarios para la vinculación de una base de datos con PHP
  • Realizar opciones de consulta, inserción, actualización y eliminación (CRUD)

Contenido

El contenido de este OVI se basa en 3 secciones donde se tomaran los temas basicos del lenguaje PHP, como la declaración de variables, hasta los mas complejos como lo es el manejo del CRUD en la base de datos
