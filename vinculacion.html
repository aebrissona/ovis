Vinculación con la DB

Requisitos de software necesarios para la conexión y administración de la base de datos con PHP.

Para trabajar PHP con una vinculación a las bases de datos necesitaremos de ciertas herramientas de software que permiten que dicha vinculación sea exitosa.

Las herramientas son:

PHP

El paquete de servidor local es el que nos proveerá de la mayoría de requisitos para poder trabajar PHP con base de datos, ya que contienen PHP, un gestor de base de datos y el área de trabajo donde se muestra lo que vamos haciendo en el código.

De manera sencilla se pueden destacar tres funciones que permiten establecer una conexión con la base de datos desde un proyecto PHP.

Mysql_connect (mysql_connect (s.f.))

Permite conectar al servidor y al gestor de base de datos, recibiendo los parámetros de servidor, usuario, contraseña. A su vez permite conectar con la base de datos usando la función mysql_select_db de la siguiente manera:

Figura 5. Ejemplo de Conexión a la DB con la Función mysql_connect.

Mysqli_connect (mysqli_connect (s.f.))

A diferencia de la función anterior este recibe un cuarto parámetro que es la base de datos permitiendo en una sola función realizar todas las conexiones necesarias para la vinculación de la siguiente manera:

Figura 5. Ejemplo de Conexión a la DB con la Función mysqli_connect.

PDO (Conexiones y su Administración (s.f.))

Las conexiones se establecen creando instancias de la clase base PDO. No importa el controlador que se utilice; siempre se usará el nombre de la clase PDO. El constructor acepta parámetros para especificar el origen de la base de datos y, opcionalmente, el nombre de usuario y la contraseña.
